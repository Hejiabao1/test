基于CNN_LSTM和脑电信号的抑郁症检测技术

1.数据集--data
	被试者在特定朗读任务下同步采集的脑电信号。其中训练数据保存在traindata文件夹中。ID1-ID22为健康对照组（label为1），ID23-ID30为抑郁症患者（label为2）。脑电为前额叶Fp1和Fp2导联信号,采样频率为250Hz。

2.数据处理及加载--dataset.py
	脑电数据为双通道数据且针对每一个患者的脑电信号并不相同，且数据较多，我们团队采用滑动窗口的算法，以250Hz采样率(FS)为基准，3s为一次采集时间来将大小不一的数据分割为一定样本数的相同时间步长数据。经过滑动窗口得到序列片段后接着循环遍历每一个数据并将其放在一个空列表中，遍历完成后将数据堆叠在一起，最后将其保存为pkl文件，至此数据预处理完成，数据以[样本数，通道数，时间步长]的尺寸格式进行接下来的操作。

3.模型--CNN_LSTM.py
	实例化模型时可根据实际需求设置隐藏层的大小、序列长度、以及是否使用双向RNN以及选择的网络类型。

4.模型训练self_train.py
	实现模型训练，并保存最优权重的模型以及模型训练过程中的表现放在depression.txt文件中。

5.模型测试test.py
	对所保存的最优权重的模型进行测试，预测训练样本标签，并打印出准确率，特异性，敏感性以及F1分数。
